## 📱 What It Does

A fully functional Android IME keyboard usable in any app. Above the QWERTY keys, an AI toolbar has two buttons:

| Button | What It Does |
|--------|-------------|
| ✨ **Improve** | Rewrites your typed text in a professional, polished tone |
| 📐 **Format** | Structures your text with proper paragraphs and punctuation |

## 🚀 Setup

1. Clone this repo
2. Get a free Gemini API key from [aistudio.google.com](https://aistudio.google.com/app/apikey)
3. Add it in `app/build.gradle` → `buildConfigField "String", "GEMINI_API_KEY", '"YOUR_KEY"'`
4. Build in Android Studio → Run on your phone
5. Settings → Keyboards → Enable **AI Keyboard**

- Kotlin · Android IME · Gemini 2.5 Flash REST API · Coroutines
